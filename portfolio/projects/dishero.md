# Dishero - Restaurant Discovery App

**Period:** May 2015 – May 2017
**Role:** Lead iOS Developer

## Overview
Pilot project from San Francisco that aggregated restaurant menus, enabling dish discovery and user-centric recommendations. Focused on dishes, not just restaurants — a unique approach to food discovery.

## Stack
- Objective-C
- CoreData
- REST APIs
- Redmine (project management)

## Key Features
- Dish-centric discovery (search by specific dishes, not just restaurants)
- User-specific recommendations
- Complex data filters for menu exploration
- CoreData for local caching and performance

## Technical Achievements
- Built REST API integrations for aggregating menu data
- Implemented complex filtering system for dish discovery
- Developed recommendation engine for user preferences
- CoreData architecture for offline functionality

## Domain
- Food tech
- Discovery and recommendation systems
- Data aggregation
